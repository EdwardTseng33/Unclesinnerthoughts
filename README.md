# 大叔心聲 — DaShu Voice

AI-powered mental health companion for Taiwanese adult males (30-50).

> 不叫你加油、不給你雞湯，就一個不會已讀你的地方。

## Tech Stack
- Pure HTML/CSS/JS (single file, zero dependencies)
- Canvas particle system
- Pre-scripted conversation tree with 50+ nodes
- Color journey system with circular hue blending

## Deploy
Static site — deploy anywhere (Vercel, Netlify, GitHub Pages).
